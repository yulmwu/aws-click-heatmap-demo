Heatmap Click Producer → Kinesis Data Streams
Click on the stage to send events via AWS SDK (browser). Demo-only.

Each click sends one record to Kinesis. Payload includes x/y, viewport, dpr, pageId, eventTime.

If your browser blocks requests due to CORS, run the same code from a backend (Next.js API route / Lambda / API Gateway).
